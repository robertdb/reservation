# reservation

SET .env file to choice your game:

USER_GOLF="myuser"
PASSWORD="mypass"
WISH_GAME={"day": "Jue", "dayNumber": "17", "month": "Mar", "times": [{"time":"07:50"}], "available": "1"}

## Run:

npm install

node golf-scraper.js
